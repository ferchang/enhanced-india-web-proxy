India Web Proxy v1.0 © 2009-2010

Web Proxy Tunnel between Web Sever and Client.
Bypass Proxy Restrictions and Get access to blocked sites.

________________________________________________________
1) Upload "index.php" on Website (PHP Support)

2) Install "Java Runtime Environment" - Download: http://www.java.com/en/download/manual.jsp

3) Run IndiaWebProxy.jar

4) Go to Settings, and Set "Web Server IP" & "File Full URL"

5) Set Internet Browser Proxy ==> Proxy: "127.0.0.1" Port:"6050"

6) "Start" IndiaWebProxy.

________________________________________________________
 For jar source: extract "javasrc.zip" of "IndiaWebProxy.jar" archive.
